### 페어 : `유스`, `푸만능`

### 실행 방법

```
// 저장소 클론
git clone -b step1 --single-branch https://github.com/turtle601/javascript-lotto-1.git

// 패키지 설치
npm i

// 어플리케이션 실행
npm run start-step1

```
